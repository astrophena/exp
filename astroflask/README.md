# `astroflask`

This is an example app for learning Flask.

It's available on
[flask.astrophena.name](https://flask.astrophena.name) and hosted on
[Fly.io](https://fly.io).

## Hacking

```sh
$ virtualenv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ ./dev
```

Open http://localhost:3000 in your browser.
